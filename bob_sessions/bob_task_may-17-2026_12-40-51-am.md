694 | **Symptoms**: Error "Failed to parse LLM response as JSON"

696 | **Solutions**:

703 | 
704 | **Symptoms**: Error "GitHub API rate limit exceeded"
705 | 
706 | **Solutions**:
707 | 1. Add `GITHUB_TOKEN` to `.env.local`
708 | 2. Reduce number of files analyzed
709 | 3. Wait for rate limit reset (check response headers)

711 | ### Issue 3: Timeout on Large Modules
712 | 
713 | **Symptoms**: Request takes >60 seconds, Vercel timeout
